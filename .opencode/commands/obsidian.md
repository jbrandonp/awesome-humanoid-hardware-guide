# Commande d'archivage Obsidian

Tu dois archiver les informations importantes de notre conversation actuelle dans mon Vault Obsidian situé ici : `C:\Users\brand\OneDrive\Documents\Obsidian Vault\IDE`

**Tes instructions :**
1. **Ne me pose aucune question** et génère le fichier immédiatement en utilisant tes outils d'écriture.
2. Choisis toi-même un nom de fichier pertinent au format `YYYY-MM-DD_Sujet.md`.
3. Rédige un résumé clair de ce qu'on vient de faire (avec le code final si applicable) au format Markdown.
4. **Où le sauvegarder ?**
    - Si ça parle de code ou d'architecture : mets-le dans le dossier `/Code`.
    - Si c'est de la théorie ou des explications : mets-le dans `/Docs`.
    - Sinon, mets-le dans `/Journal`.
5. Confirme-moi simplement que c'est fait avec le chemin du fichier créé.
