# 01.5 — Ajouter ts-loader et webpack-node-externals

**Réf** : `260426.md:290` (C6)  
**Priorité** : 🔴 P1 — BLOQUANT BUILD (webpack échoue)  
**Durée estimée** : 2 min  
**Dépend de** : P0

---

## Contexte

`apps/api/webpack.config.js` utilise `ts-loader` et `webpack-node-externals` mais ces packages ne sont **pas dans `package.json`**. Le build webpack de l'API échoue.

---

## Fichier concerné

```
package.json (racine)
```

---

## Action

Ajouter dans `devDependencies` :

```json
"ts-loader": "^9.5.0",
"webpack-node-externals": "^3.0.0"
```

Puis lancer `npm install --legacy-peer-deps`.

---

## Vérification

```bash
node -e "require('ts-loader')" && echo "ts-loader OK"
node -e "require('webpack-node-externals')" && echo "webpack-node-externals OK"
```

---

## Risques

- **Faible** : Packages standards, bien maintenus
- Si conflit de version → vérifier la compatibilité avec la version de webpack utilisée
