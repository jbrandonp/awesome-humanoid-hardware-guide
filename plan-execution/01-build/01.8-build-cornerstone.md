# 01.8 — Build librairie cornerstone-integration

**Réf** : `260426.md:951`  
**Priorité** : 🔴 P1 — SÉQUENTIEL (dépend de 01.6)  
**Durée estimée** : 3 min

---

## Contexte

La librairie `cornerstone-integration` est utilisée par le frontend desktop pour l'affichage DICOM. Doit être buildée.

---

## Fichier concerné

```
libs/cornerstone-integration/
```

---

## Action

```bash
npx nx build cornerstone-integration
```

---

## Vérification

```bash
ls libs/cornerstone-integration/dist/index.js && echo "OK" || echo "FAIL"
```

---

## Risques

- **Moyen** : `cornerstone-state.service.ts` utilise `(window as any).cornerstoneTools` → crash hors navigateur (LB7, `260426.md:665`)
- Peut nécessiter un mock pour le build serveur
