# 01.1 — Supprimer apps/api/src/app/package.json

**Réf** : `260426.md:286` (C2)  
**Priorité** : 🔴 P1 — BLOQUANT BUILD  
**Durée estimée** : 1 min  
**Dépend de** : P0

---

## Contexte

Un `package.json` dupliqué existe dans `apps/api/src/app/`. Il a le même `"name": "@systeme-sante/source"` que le package racine, ce qui **casse npm workspace** et la résolution des dépendances.

---

## Fichier concerné

```
apps/api/src/app/package.json
```

---

## Action

```bash
rm apps/api/src/app/package.json
```

---

## Avant/Après

**Avant** : Fichier parasite présent → npm workspace confondu  
**Après** : Fichier supprimé → npm resolve correctement

---

## Vérification

```bash
ls apps/api/src/app/package.json
# Doit retourner : No such file or directory
```

---

## Risques

- **Faible** : Fichier parasite, copie partielle du package.json racine
- Si contenu utile → vérifier avant suppression : `cat apps/api/src/app/package.json`
