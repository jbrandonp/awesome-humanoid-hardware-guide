# 01.6 — Build librairie models

**Réf** : `260426.md:945`  
**Priorité** : 🔴 P1 — SÉQUENTIEL (dépend de 01.5)  
**Durée estimée** : 3 min  
**Dépend de** : 01.4, 01.5

---

## Contexte

La librairie `models` contient les schémas Prisma, WatermelonDB, et Zod. Elle doit être buildée avant les autres librairies et apps qui en dépendent.

---

## Fichier concerné

```
libs/models/
```

---

## Action

```bash
npx nx build models
```

---

## Vérification

```bash
ls libs/models/dist/index.js && echo "OK" || echo "FAIL"
```

---

## Risques

- **Moyen** : Si le build échoue → vérifier que Prisma generate a été exécuté
- Les dépendances de models (Prisma, WatermelonDB) doivent être installées
