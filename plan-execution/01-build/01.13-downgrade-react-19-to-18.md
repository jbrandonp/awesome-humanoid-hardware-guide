# 01.13 — Downgrade React 19 → 18

**Réf** : `260426.md:285` (C1) + `260426.md:806-816` + `260426.md:971-984`  
**Priorité** : 🔴 P1 — BLOQUANT BUILD (Next.js 14 incompatibilité)  
**Durée estimée** : 10 min  
**Dépend de** : P0

---

## Contexte

React 19 est **incompatible** avec Next.js 14. Deux options :
1. Upgrade Next.js 14 → 15 (3-5h, breaking changes majeures)
2. **Downgrade React 19 → 18** (10 min, option retenue)

---

## Fichiers concernés

1. `package.json` (racine) — dependencies, devDependencies, overrides
2. `apps/portail-pwa/package.json` — dependencies

---

## Action

### 1. Racine — `dependencies`

| Package | Avant | Après |
|---------|-------|-------|
| `react` | `^19.1.0` | `^18.3.1` |
| `react-dom` | `^19.1.0` | `^18.3.1` |

### 2. Racine — `devDependencies`

| Package | Avant | Après |
|---------|-------|-------|
| `@types/react` | `^19.1.0` | `^18.3.0` |
| `@types/react-dom` | `^19.0.0` | `^18.3.0` |

### 3. Racine — `overrides`

**Supprimer** le bloc `overrides` qui force React 19 (lignes ~176-181).

### 4. Portail-PWA — `dependencies`

| Package | Avant | Après |
|---------|-------|-------|
| `react` | `^19.1.0` | `^18.3.1` |
| `react-dom` | `^19.1.0` | `^18.3.1` |

---

## Vérification

```bash
npm install --legacy-peer-deps
npx nx build portail-pwa
```

---

## Risques

- **Moyen** : Certains composants peuvent utiliser des APIs React 19 (ex: `useCameraPermissions` destructuring à 2 vs 3 éléments — F14, `260426.md:181`)
- Si erreurs → identifier les APIs React 19 utilisées et adapter
- Alternative si trop de breaking changes : upgrade Next.js 14 → 15
