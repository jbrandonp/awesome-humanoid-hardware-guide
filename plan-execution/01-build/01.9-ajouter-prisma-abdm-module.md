# 01.9 — Ajouter PrismaModule dans abdm.module.ts

**Réf** : `260426.md:601` (AB1) + `260426.md:958`  
**Priorité** : 🔴 P1 — BLOQUANT BUILD (DI crash)  
**Durée estimée** : 2 min  
**Dépend de** : P0

---

## Contexte

`AbdmService` et `InventoryService` dépendent de `PrismaService` mais le module ABDM n'importe pas `PrismaModule`. NestJS DI crash au démarrage.

---

## Fichier concerné

```
apps/api/src/app/abdm/abdm.module.ts
```

---

## Action

Dans le tableau `imports` du module, ajouter `PrismaModule` :

**Avant** :
```ts
@Module({
  imports: [/* ... */],
  providers: [AbdmService, InventoryService],
  exports: [AbdmService, InventoryService],
})
```

**Après** :
```ts
@Module({
  imports: [/* ... */, PrismaModule],
  providers: [AbdmService, InventoryService],
  exports: [AbdmService, InventoryService],
})
```

---

## Vérification

```bash
npx nx typecheck api
```

---

## Risques

- **Faible** : Ajout standard, pas de breaking change
- Vérifier que `PrismaModule` est bien exporté depuis son module
