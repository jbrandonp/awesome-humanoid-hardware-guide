# 01.10 — Ajouter PrismaModule dans ocr.module.ts

**Réf** : `260426.md:609` (AB10) + `260426.md:962`  
**Priorité** : 🔴 P1 — BLOQUANT BUILD (DI crash)  
**Durée estimée** : 2 min  
**Dépend de** : P0

---

## Contexte

Le module OCR utilise `PrismaService` mais n'importe pas `PrismaModule`. NestJS DI crash au démarrage.

---

## Fichier concerné

```
apps/api/src/app/ocr/ocr.module.ts
```

---

## Action

Dans le tableau `imports` du module, ajouter `PrismaModule`.

**Avant** :
```ts
@Module({
  imports: [/* ... */],
  // ...
})
```

**Après** :
```ts
@Module({
  imports: [/* ... */, PrismaModule],
  // ...
})
```
