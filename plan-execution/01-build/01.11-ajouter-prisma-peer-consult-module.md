# 01.11 — Ajouter PrismaModule dans peer-consult.module.ts

**Réf** : `260426.md:608` (AB9) + `260426.md:964`  
**Priorité** : 🔴 P1 — BLOQUANT BUILD (DI crash)  
**Durée estimée** : 2 min  
**Dépend de** : P0

---

## Contexte

Le module Peer Consult utilise `PrismaService` mais n'importe pas `PrismaModule`. NestJS DI crash au démarrage.

---

## Fichier concerné

```
apps/api/src/app/peer-consult/peer-consult.module.ts
```

---

## Action

Dans le tableau `imports` du module, ajouter `PrismaModule`.

**Avant** :
```ts
@Module({
  imports: [/* ... */],
  // ...
})
```

**Après** :
```ts
@Module({
  imports: [/* ... */, PrismaModule],
  // ...
})
```

---

## Vérification

```bash
npx nx typecheck api
```

## Risques

- **Faible** : Ajout standard
