  // ...

## Vérification

```bash
curl http://localhost:3000/api/remote-patient-monitoring/vitals
# Doit retourner 401 Unauthorized

curl -H "Authorization: Bearer <token>" http://localhost:3000/api/remote-patient-monitoring/vitals
# Doit retourner 200
```
