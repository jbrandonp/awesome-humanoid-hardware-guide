# 04.4 — Ajouter authentification sur kiosk.gateway.ts

**Réf** : `260426.md:73` (A10) + `260426.md:1068-1069`  
**Priorité** : 🟠 P4 — CRITIQUE SÉCURITÉ  
**Durée estimée** : 10 min  
**Dépend de** : P1

---

## Contexte

Le WebSocket Kiosk n'a **aucune authentification**. N'importe qui peut se connecter et recevoir les données patients en temps réel.

---

## Fichier concerné

```
apps/api/src/app/kiosk/kiosk.gateway.ts
```

---

## Action

Ajouter la vérification JWT dans `handleConnection()` :

```ts
@WebSocketGateway()
export class KioskGateway implements OnGatewayConnection {
  @WebSocketServer() server: Server;

  async handleConnection(client: Socket) {
    const token = client.handshake.auth.token
      || client.handshake.query.token as string;
    
    if (!token) {
      client.disconnect();
      return;
    }

    try {
      const payload = this.jwtService.verify(token);
      client.data.user = payload;
    } catch {
      client.disconnect();
    }
  }
}
```

Ajouter `@UseGuards(AuthGuard('jwt'))` sur les handlers de messages.

---

## Vérification

```bash
# Connexion sans token → doit être rejetée
websocat ws://localhost:3000/kiosk
# → déconnecté immédiatement

# Connexion avec token valide
websocat ws://localhost:3000/kiosk?token=<jwt_valide>
# → connecté
```

---

## Risques

- **Moyen** : Impacte les kiosks en production si le token n'est pas encore configuré côté client
- Vérifier que le client PWA envoie bien le token
