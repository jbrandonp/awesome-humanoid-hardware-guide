# 08.4 — Remplacer les @ts-ignore par des corrections réelles

**Réf** : `260426.md:1222-1224`  
**Priorité** : 🟢 P8 — QUALITÉ  
**Durée estimée** : 10 min  
**Dépend de** : P1

---

## Contexte

4 fichiers utilisent `@ts-ignore` pour masquer des erreurs TypeScript. Il faut les remplacer par des corrections de types réelles.

---

## Fichiers et corrections

| Fichier | Ligne | Correction |
|---------|-------|------------|
| `RPPGBiosensor.tsx` | 3 | Typer correctement l'import Expo Camera |
| `BarcodeAdministration.tsx` | 3 | Même problème Expo Camera |
| `HardwareStatusIndicator.tsx` | 25 | Déclarer l'interface `__TAURI_INTERNALS__` au lieu de `@ts-ignore` |
| `app.tsx` | 21 | Typer correctement l'erreur Tauri au lieu de `as string` |

---

## Exemple — `HardwareStatusIndicator.tsx:25`

**Avant** :
```ts
// @ts-ignore
const isTauri = !!window.__TAURI_INTERNALS__;
```

**Après** :
```ts
declare global {
  interface Window {
    __TAURI_INTERNALS__?: unknown;
  }
}
const isTauri = !!window.__TAURI_INTERNALS__;
```
