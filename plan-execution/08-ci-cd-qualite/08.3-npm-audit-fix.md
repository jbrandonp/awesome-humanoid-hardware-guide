# 08.3 — npm audit fix

**Réf** : `260426.md:1219-1220`  
**Priorité** : 🟢 P8 — QUALITÉ  
**Durée estimée** : 10 min  
**Dépend de** : P1

---

## Contexte

75 vulnérabilités npm (1 critique, 22 hautes). Après toutes les corrections de dépendances (P1), il faut auditer et corriger les vulnérabilités restantes.

---

## Action

```bash
# Voir les vulnérabilités
npm audit

# Corriger automatiquement (non-breaking)
npm audit fix

# Si des vulnérabilités nécessitent des breaking changes
npm audit fix --force  # ATTENTION : peut casser le build
```

---

## Vérification

```bash
npm audit --audit-level=moderate
# Aucune vulnérabilité modérée ou plus
```

---

## Risques

- **Élevé** : `npm audit fix --force` peut introduire des breaking changes
- Toujours tester le build complet après : `npx nx run-many --target=build --all`
