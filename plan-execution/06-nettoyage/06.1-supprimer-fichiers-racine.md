# 06.1 — Supprimer les fichiers parasites de la racine

**Réf** : `260426.md:1126-1130`  
**Priorité** : 🟡 P6 — NETTOYAGE  
**Durée estimée** : 2 min  
**Dépend de** : P1

---

## Contexte

La racine du projet est encombrée de fichiers de log, debug, et scripts temporaires qui polluent le repo.

---

## Fichiers à supprimer

```
api-error.txt
api-output.txt
api.err
api2.err
api3.err
api4.err
api2.txt
api3.txt
api4.txt
mock-server.err
stderr.txt
stdout.txt
tsclist.txt
test5.js
test-server.js
null
```

---

## Action

```bash
rm api-error.txt api-output.txt api.err api2.err api3.err api4.err \
   api2.txt api3.txt api4.txt mock-server.err stderr.txt stdout.txt \
   tsclist.txt test5.js test-server.js null
```

---

## Vérification

```bash
ls api-* stderr.txt stdout.txt null test5.js test-server.js 2>&1
# Tous doivent retourner "No such file or directory"
```

---

## Risques

- **Faible** : Fichiers de debug, pas utilisés en production
- Vérifier que `null` n'est pas utilisé par un script
