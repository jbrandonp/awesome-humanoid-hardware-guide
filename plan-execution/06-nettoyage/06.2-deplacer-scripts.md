# 06.2 — Déplacer les scripts vers leurs dossiers appropriés

**Réf** : `260426.md:1132-1135`  
**Priorité** : 🟡 P6 — NETTOYAGE  
**Durée estimée** : 5 min  
**Dépend de** : P1

---

## Contexte

Des scripts utiles sont éparpillés à la racine. Ils doivent être déplacés dans des dossiers appropriés.

---

## Actions

```bash
# Scripts tools
mv mock-server.js scripts/tools/
mv test-db.js scripts/tools/
mv start-api.js scripts/tools/

# Scripts Kubernetes
mv create-secrets.ps1 infra/kubernetes/
mv generate-secrets.ps1 infra/kubernetes/
```

---

## Vérification

```bash
ls scripts/tools/mock-server.js scripts/tools/test-db.js scripts/tools/start-api.js
ls infra/kubernetes/create-secrets.ps1 infra/kubernetes/generate-secrets.ps1
```

---

## Risques

- **Moyen** : Des scripts CI ou npm scripts peuvent référencer ces fichiers avec l'ancien chemin
- Vérifier `package.json` scripts et `.github/workflows/` pour mettre à jour les chemins
