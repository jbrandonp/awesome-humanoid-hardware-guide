# 00.1 — Installer Visual Studio Build Tools 2022

**Réf** : `260426.md:338` (Phase 0 action 1)  
**Priorité** : 🔴 P0 — PRÉREQUIS ABSOLU  
**Durée estimée** : 20 min

---

## Contexte

Les modules Rust/Tauri nécessitent un compilateur C++ pour compiler les crates natives. Sans VS Build Tools, `cargo build` échoue.

---

## Action

1. Télécharger VS Build Tools 2022 : https://visualstudio.microsoft.com/downloads/#build-tools-for-visual-studio-2022
2. Lancer l'installateur
3. Dans l'onglet "Workloads", cocher **"Desktop development with C++"**
4. Installer
5. Redémarrer la machine si demandé

---

## Vérification

```bash
where cl.exe
```
Doit retourner un chemin valide (ex: `C:\Program Files\Microsoft Visual Studio\2022\BuildTools\VC\Tools\MSVC\...\bin\Hostx64\x64\cl.exe`)

---

## Risques

- **Faible** : Installation standard Microsoft
- Si déjà installé → passer directement à la vérification
