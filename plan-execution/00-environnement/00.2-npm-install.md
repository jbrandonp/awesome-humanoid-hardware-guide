# 00.2 — npm install --legacy-peer-deps

**Réf** : `260426.md:338` (Phase 0 action 2)

## Contexte

Le projet a des conflits de peer dependencies (React 19 vs Next.js 14, Jest 30 vs ts-jest 29, etc.) qui empêchent un `npm install` normal. `--legacy-peer-deps` contourne ces conflits temporairement avant les corrections en P1.

---

## Action

```bash
npm install --legacy-peer-deps
```

---

## Vérification

```bash
node -e "require('@prisma/client')"
npx nx --version
```

Si `gl` package échoue :
```bash
npm ls gl
# Identifier la source et l'overrider dans package.json
```

Si `brain.js` cause problème → ajouter dans `overrides` ou supprimer temporairement.

---

## Risques

- **Moyen** : Si `node_modules/` est corrompu, supprimer `node_modules/` et `package-lock.json` puis relancer
- Les conflits de peer dependencies seront résolus définitivement en P1 (01.4, 01.13)
