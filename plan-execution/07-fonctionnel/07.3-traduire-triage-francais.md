# 07.3 — Traduire les mots-clés de triage en français

**Réf** : `260426.md:72` (A9) + `260426.md:852-859` + `260426.md:1174-1177`  
**Priorité** : 🟡 P7 — CRITIQUE FONCTIONNEL (impact clinique)  
**Durée estimée** : 5 min  
**Dépend de** : P1

---

## Contexte

Les mots-clés de triage sont en **anglais** (`'chest pain'`, `'stroke'`, `'suicidal'`) dans un système conçu pour le **français**. Les patients avec ces symptômes ne sont jamais détectés → mal classés (niveau de gravité sous-estimé).

---

## Fichier concerné

```
apps/api/src/app/triage/triage.algorithm.ts:23-29
```

---

## Action

**Avant** :
```ts
const CRITICAL_KEYWORDS = ['chest pain', 'stroke', 'suicidal'];
```

**Après** :
```ts
const CRITICAL_KEYWORDS = [
  'douleur thoracique', 'douleur poitrine', 'douleur sternum',
  'avc', 'accident vasculaire', 'attaque cérébrale',
  'suicidaire', 'idées noires', 'tentative suicide',
];
```

---

## Vérification

```bash
# Test : patient avec "douleur thoracique" → doit être classé ESI 2 (urgent)
npx nx test triage
```

---

## Risques

- **Moyen** : Si des patients utilisent des termes anglais, ils seront ratés
- Idéalement : supporter les deux langues (français + anglais) + synonymes
