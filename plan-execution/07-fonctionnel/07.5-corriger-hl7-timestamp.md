# 07.5 — Corriger le timestamp HL7 : supprimer le T ISO

**Réf** : `260426.md:667` (LB9) + `260426.md:1182-1184`  
**Priorité** : 🟡 P7 — FONCTIONNEL  
**Durée estimée** : 3 min  
**Dépend de** : P1

---

## Contexte

Le timestamp HL7 doit être au format `YYYYMMDDHHMMSS`. Le code supprime `-` et `:` mais **pas le `T`** du format ISO 8601 (`2026-04-26T14:30:00`). Résultat : `20260426T143000` → format HL7 invalide.

---

## Fichier concerné

```
apps/api/src/app/hl7/hl7-mllp.controller.ts:180
```

---

## Action

**Avant** :
```ts
const hl7Timestamp = new Date().toISOString().replace(/[-:]/g, '');
```

**Après** :
```ts
const hl7Timestamp = new Date().toISOString().replace(/[-:T]/g, '').substring(0, 14);
```

---

## Vérification

```bash
node -e "console.log(new Date().toISOString().replace(/[-:T]/g, '').substring(0, 14))"
# Doit afficher : 20260426143000 (14 chiffres, pas de T)
```

---

## Risques

- **Faible** : Correction de 1 caractère dans une regex
