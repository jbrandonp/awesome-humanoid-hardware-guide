# 07.8 — Corriger la pression artérielle OCR (enlever "...")

**Réf** : `260426.md:610` (AB11) + `260426.md:1197-1198`  
**Priorité** : 🟡 P7 — FONCTIONNEL  
**Durée estimée** : 10 min  
**Dépend de** : P1

---

## Contexte

La pression artérielle extraite par OCR est stockée comme `"120/..."` (avec `...` littéral) au lieu de `"120/80"`. Les données cliniques sont **polluées**.

---

## Fichier concerné

```
apps/api/src/app/ocr/ocr.controller.ts:58-60
```

---

## Action

**Avant** (valeur factice) :
```ts
const bloodPressure = `${systolic}/...`;
```

**Après** (extraction réelle) :
```ts
// Extraire la diastolique depuis le texte OCR
const bpMatch = ocrText.match(/(\d{2,3})\s*\/\s*(\d{2,3})/);
const diastolic = bpMatch ? bpMatch[2] : null;
const bloodPressure = diastolic ? `${systolic}/${diastolic}` : `${systolic}/?`;
```

---

## Vérification

```bash
# Test avec un texte OCR contenant "120/80"
# → doit retourner "120/80", pas "120/..."
npx nx test ocr
```

---

## Risques

- **Moyen** : Si l'OCR ne trouve pas de diastolique, on met `?` (mieux que `...`)
- Idéalement : flagger la valeur comme incomplète pour révision manuelle
