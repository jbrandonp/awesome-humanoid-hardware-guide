## Contexte

`intelligence.service.ts` a **seulement 2 paires** d'interactions médicamenteuses hardcodées. Tout le reste retourne `"SAFE"`. En production, des milliers de paires sont nécessaires. Un médecin peut prescrire une combinaison dangereuse non détectée.

## Action

### 1. Créer une table `drug_interactions` (Prisma)

```prisma
model DrugInteraction {
  id          String   @id @default(uuid())
  drugA       String
  drugB       String
  severity    Severity
  description String
  @@index([drugA, drugB])
}

enum Severity {
  LOW
  MODERATE
  HIGH
  CONTRAINDICATED
}
```

### 2. Charger depuis la DB

```ts
async checkInteractions(medications: string[]): Promise<Interaction[]> {
  const interactions = await this.prisma.drugInteraction.findMany({
    where: {
      drugA: { in: medications },
      drugB: { in: medications },
    },
  });
  return interactions.map(i => ({
    drugs: [i.drugA, i.drugB],
    severity: i.severity,
    description: i.description,
  }));
}
```

### 3. Seeder la table avec des données réelles

Importer une base type DrugBank, OpenFDA, ou ANSM (base française).
