## Contexte

`processSmartPenInk()` crée un **nouveau** `Y.Doc()` à chaque stroke, écrasant tous les strokes précédents. Le CRDT n'est pas implémenté. Il faut lire l'état existant, appliquer le merge Yjs, puis sauvegarder.

## Fichier concerné

```
apps/api/src/app/iot/iot.service.ts:322-350
```

## Action

**Avant** (écrase tout) :
```ts
const doc = new Y.Doc();
// insère le stroke
const state = Y.encodeStateAsUpdate(doc);
await this.prisma.visit.update({
  where: { id: visitId },
  data: { notes: Buffer.from(state) }
});
```

**Après** (merge CRDT) :
```ts
const existingVisit = await this.prisma.visit.findUnique({
  where: { id: visitId },
  select: { notes: true }
});

const doc = new Y.Doc();
// Appliquer l'état existant AVANT le nouveau stroke
if (existingVisit?.notes) {
  Y.applyUpdate(doc, new Uint8Array(existingVisit.notes));
}

// Insérer le nouveau stroke
const yText = doc.getText('strokes');
yText.insert(yText.length, JSON.stringify(newStroke));

// Encoder et sauvegarder
const newState = Y.encodeStateAsUpdate(doc);
await this.prisma.visit.update({
  where: { id: visitId },
  data: { notes: Buffer.from(newState) }
});
```
