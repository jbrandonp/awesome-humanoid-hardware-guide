# 02.1 — Lancer les tests des librairies

**Réf** : `260426.md:997`  
**Priorité** : 🔴 P2 — VALIDATION  
**Durée estimée** : 5 min  
**Dépend de** : P1 complète

---

## Contexte

Une fois les librairies buildées (P1), lancer leurs tests pour valider l'intégrité avant de continuer.

---

## Action

```bash
npx nx test models
npx nx test insurance-engine
npx nx test cornerstone-integration
```

---

## Vérification

Tous les tests doivent passer. Si échec → corriger avant de continuer.

---

## Risques

- **Élevé** : Les tests peuvent échouer à cause des changements de P1 (Jest downgrade, React downgrade)
- Si `cornerstone-integration` échoue à cause de `window` → mocker les APIs navigateur
