# 02.4 — Lancer les tests mobile

**Réf** : `260426.md:1008`  
**Priorité** : 🔴 P2 — VALIDATION  
**Durée estimée** : 5 min  
**Dépend de** : P1

---

## Contexte

Lancer la suite de tests du frontend mobile pour identifier les tests cassés après les corrections P1 (notamment React 18 downgrade).

---

## Action

```bash
npx nx test mobile
```

---

## Problèmes anticipés

- `@types/react` 18 vs 19 — différences de types
- `useCameraPermissions` destructuring (F14, `260426.md:181`) — Expo peut retourner 2 ou 3 éléments selon la version
- `Buffer` non disponible en React Native (FR1, `260426.md:623`)

---

## Vérification

Tous les tests doivent passer. Identifier les échecs et corriger.

---

## Risques

- **Moyen** : Tests mobile souvent fragiles à cause des dépendances natives
