# 02.2 — Corriger nursing-station.service.spec.ts @ts-expect-error

**Réf** : `260426.md:246` (T11) + `260426.md:999-1001`  
**Priorité** : 🔴 P2 — BLOQUANT TESTS  
**Durée estimée** : 3 min  
**Dépend de** : P1

---

## Contexte

Deux directives `@ts-expect-error` sont placées sur des variables qui sont **utilisées**. TypeScript génère une erreur "Unused @ts-expect-error directive", ce qui fait échouer la compilation.

---

## Fichier concerné

```
nursing-station.service.spec.ts:10,13
```

---

## Action

**Avant** :
```ts
// @ts-expect-error unused
const mockPrismaService = { ... };
// @ts-expect-error unused
const service = new NursingStationService(mockPrismaService);
```

**Après** (supprimer les directives) :
```ts
const mockPrismaService = { ... };
const service = new NursingStationService(mockPrismaService);
```

npx nx test nursing-station
