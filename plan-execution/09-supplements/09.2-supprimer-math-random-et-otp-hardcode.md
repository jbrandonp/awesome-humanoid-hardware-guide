# 09.2 — Supprimer Math.random() et OTP hardcodé de abdm.service.ts

**Réf** : `260426.md:97` (A29) + `260426.md:602` (AB3)  
**Priorité** : 🟠 MANQUANT — CODE DE TEST EN PRODUCTION + SÉCURITÉ  
**Durée estimée** : 10 min  
**Dépend de** : P1

---

## Contexte

- **A29** : `Math.random() < 0.2` simule 20% d'échec API → **code de test en production**
- **AB3** : OTP hardcodé `'123456'` → bypass complet de la vérification OTP

---

## Fichier concerné

```
apps/api/src/app/abdm/abdm.service.ts
```

---

## Action

### 1. Supprimer Math.random() (ligne ~103)

**Avant** :
```ts
if (Math.random() < 0.2) {
  throw new Error('Simulated ABDM failure');
}
```

**Après** :
```ts
// Supprimé — pas de simulation en production
```

### 2. Remplacer OTP hardcodé

**Avant** (ligne ~103) :
```ts
if (otp !== '123456') {
  throw new UnauthorizedException('Invalid OTP');
}
```

**Après** :
```ts
const storedOtp = await this.redis.get(`otp:${sessionId}`);
if (!storedOtp || otp !== storedOtp) {
  throw new UnauthorizedException('Invalid OTP');
}
await this.redis.del(`otp:${sessionId}`);
```
