# 03.5 — Ajouter output: 'standalone' dans next.config.js

**Réf** : `260426.md:293` (C9) + `260426.md:1039-1042`  
**Priorité** : 🔴 P3 — BLOQUANT DOCKER BUILD (PWA)  
**Durée estimée** : 2 min  
**Dépend de** : P1

---

## Contexte

`Dockerfile.web` copie `.next/standalone` mais `next.config.js` n'a pas `output: 'standalone'`. Next.js ne génère pas le dossier standalone → le build Docker échoue.

---

## Fichier concerné

```
apps/portail-pwa/next.config.js
```

---

## Action

Ajouter dans l'objet `nextConfig` :

**Avant** :
```js
const nextConfig = {
  // ...
};
```

**Après** :
```js
const nextConfig = {
  output: 'standalone',
  // ...
};
```

---

## Vérification

```bash
npx nx build portail-pwa
ls apps/portail-pwa/.next/standalone/ && echo "OK" || echo "FAIL"
```

---

## Risques

- **Faible** : Option standard Next.js
- Peut augmenter la taille du build
