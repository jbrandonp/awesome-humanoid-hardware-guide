# 03.2 — Corriger docker-compose.yml build context

**Réf** : `260426.md:295` (C11) + `260426.md:1024-1027`  
**Priorité** : 🔴 P3 — BLOQUANT DOCKER BUILD  
**Durée estimée** : 2 min  
**Dépend de** : P1

---

## Contexte

Le `build.context` du service API pointe vers `..` (soit `infra/`), mais doit pointer vers la racine du projet (où se trouvent `package.json`, `nx.json`, etc.).

---

## Fichier concerné

```
infra/docker/docker-compose.yml
```

---

## Action

**Avant** :
```yaml
services:
  api:
    build:
      context: ..
```

**Après** :
```yaml
services:
  api:
    build:
      context: ../..
```

---

## Vérification

```bash
docker compose -f infra/docker/docker-compose.yml config --no-color | grep context
```

---

## Risques

- **Moyen** : Vérifier que le chemin `../..` existe bien et contient le Dockerfile référencé
