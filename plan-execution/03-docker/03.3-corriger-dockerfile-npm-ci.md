# 03.3 — Corriger Dockerfiles : --only=production → --omit=dev

**Réf** : `260426.md:291` (C7) + `260426.md:1029-1032`  
**Priorité** : 🔴 P3 — BLOQUANT DOCKER BUILD  
**Durée estimée** : 3 min  
**Dépend de** : P1

---

## Contexte

`npm ci --only=production` est un flag **supprimé** dans npm 9+. Le build Docker échoue. Le remplacement moderne est `npm ci --omit=dev`.

---

## Fichiers concernés

```
infra/docker/Dockerfile.api:24
infra/docker/Dockerfile.web:29
```

---

## Action

Dans les deux fichiers :

**Avant** :
```dockerfile
RUN npm ci --only=production
```

**Après** :
```dockerfile
RUN npm ci --omit=dev
```

---

## Vérification

```bash
grep -rn "only=production" infra/docker/
# Ne doit rien retourner
```

---

## Risques

- **Faible** : Changement standard npm 9+
