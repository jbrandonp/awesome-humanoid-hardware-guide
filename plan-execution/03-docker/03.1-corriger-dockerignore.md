# 03.1 — Corriger .dockerignore : ne pas exclure package-lock.json

**Réf** : `260426.md:640` (INF1) + `260426.md:1018-1022`  
**Priorité** : 🔴 P3 — BLOQUANT DOCKER BUILD  
**Durée estimée** : 2 min  
**Dépend de** : P1

---

## Contexte

`.dockerignore` contient `*.lock` qui exclut **tous** les fichiers lock, y compris `package-lock.json`. `npm ci` exige `package-lock.json` → le build Docker échoue.

---

## Fichier concerné

```
.dockerignore:13
```

---

## Action

**Avant** :
```
*.lock
```

**Après** :
```
# *.lock  ← supprimée
!package-lock.json
```

---

## Vérification

```bash
grep -n "lock" .dockerignore
# Ne doit PAS contenir *.lock sans exception
```

---

## Risques

- **Faible** : Correction ciblée
- Autres fichiers `.lock` (yarn.lock, pnpm-lock.yaml) peuvent rester exclus
